Información sobre el tamaño de papel

Esta aplicación está configurada para generar etiquetas en el formato exacto de 100mm x 65mm. Asegúrate de tener los siguientes puntos en cuenta:

  • Cada etiqueta se imprimirá en una página separada.
  • Tu impresora debe soportar este tamaño de papel para una impresión correcta.
  • Configura en tu impresora el tamaño personalizado 100mm x 65mm.
  • El PDF generado tendrá el tamaño exacto de las etiquetas sin márgenes adicionales.

Consejos para configurar tu impresora

1. En Windows: Al imprimir, selecciona "Propiedades de la impresora" → "Configuración de página" → "Tamaño de papel personalizado".

2. Alimentación de papel: Asegúrate de tener el rollo de etiquetas correctamente instalado en tu impresora térmica.

3. Calidad de impresión: Ajusta la densidad/calidad de impresión según las necesidades de tus etiquetas.

4. Prueba: Se recomienda hacer una impresión de prueba antes de imprimir grandes cantidades.
